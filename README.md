El siguienteproyecto hace referencia al desarrollo del siguiente sistema: de Inventario y Ventas 

Este sistema tendra los modulos de compras y ventas y por ello se podra administrar el inventario.Dicha gestion de inventario administra las entradas de productos (compras) salida de productos (ventas) y se podran generar reportes de inventario por stock
